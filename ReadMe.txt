Чтобы убрать ошибку 
Ошибка	C2039	"create_directory": не является членом "std".
нужно поменять стандарт языка с++ на Стандарт ISO C++17 (std:c++17)
Ошибка с исключением 
нужно добавить аргументы командной строки для каждой конфигурации x86 x64 debug release	
Аргументы командной строки для client.exe: 127.0.0.1 5555 6000 test.txt 500
Аргументы командной строки для server.exe: 127.0.0.1 5555 temp
